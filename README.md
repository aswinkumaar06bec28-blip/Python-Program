# Basic Python Programs 🐍

A collection of simple, standalone Python scripts designed to demonstrate fundamental programming concepts. 

## 🚀 Overview

This repository contains basic Python programs that focus on core logic, standard data structures, and basic algorithms. All scripts are designed to be easily readable and runnable straight out of the box, making it a great reference for Python fundamentals.

## 🛠️ Built With
* **Python 3.x**
* **Standard Library Only** (No external dependencies or `pip install` required)

## 📂 Contents

Here is a quick look at what you will find in this repository:

| Script Name | Description | Concepts Covered |
| :--- | :--- | :--- |
| `hello_world.py` | A basic introductory script. | Syntax, `print()` |
| `calculator.py` | A simple terminal-based calculator. | Functions, user input, math |
| `number_guess.py` | A basic number guessing game. | `while` loops, conditionals, `random` |
| `string_utils.py` | Scripts to reverse strings and count vowels. | String manipulation, `for` loops |
